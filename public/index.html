Options

Process modes

Single session
Serves exactly one remote session for this working directory.
Shared directory
Allows several sessions in the same directory. Concurrent edits can conflict.
Git worktrees
Creates an isolated Git worktree for each session. The directory must be a Git repository.

Permission modes

Use Claude settings
Does not override the permission mode configured for Claude Code.
Ask before changes
Allows reads, but asks before edits and commands that require approval.
Accept edits
Automatically accepts file edits and common filesystem operations, while other actions can still ask.
Plan only
Lets Claude inspect and plan without editing the project until you change modes.

Remote Control does not offer Auto or Bypass permission modes.
